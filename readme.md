## Site Notícias Cidade

Projeto desenvolvido na "Seção 5: CSS 3 Intermediário & Avançado" do curso "Desenvolvimento Web Completo 2022".
Link para visualização: https://monicaquintal.github.io/NoticiasCidade/

### Conteúdo:
- Aula 01: Criação do topo da página;
- Aula 02: Criação da barra de navegação e das demais páginas;
- Aula 03: Layout em colunas (parte 01);
- Aula 04: Layout em colunas (parte 02) e caixa;
- Aula 05: Barra lateral;
- Aula 06: Área de conteúdos;
